# 📊 Optimización de catálogo tipo Netflix con Machine Learning

## 🎯 Objetivo
Analizar el catálogo de Netflix y construir modelos predictivos para identificar patrones en el contenido.

## 📊 Dataset
Dataset de Netflix con información de títulos, géneros, países y duración.

## 🔍 Proceso
- Limpieza de datos
- Análisis exploratorio (EDA)
- Feature engineering

## 🤖 Modelos
- Logistic Regression
- Random Forest

## 📈 Resultados
- AUC: 0.67
- Comparación de modelos

## 🧠 Insights
- Identificación de patrones en géneros
- Tendencias en producción por país

## 🚀 Tecnologías
- Python
- pandas
- scikit-learn
- matplotlib
